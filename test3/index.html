Скачивание видео через VPS backend

Скачать видео с Rutube за пару кликов

Вставляешь ссылку на видео с Rutube, нажимаешь кнопку, backend на VPS скачивает ролик и сразу отдаёт файл в твой браузер. Никаких сторонних веб-сервисов, только твой frontend, твой VPS и локальная загрузка в браузер.

> frontend: GitHub Pages

> backend: VPS downloader

> engine: yt-dlp + ffmpeg

> result: файл улетает в загрузки браузера

Поддерживаются только ссылки rutube.ru. Если Rutube изменит схему выдачи, backend можно будет обновить отдельно, не ломая сайт.

Вставь ссылку и нажми «Скачать»

Если backend доступен и ссылка валидная, браузер начнёт скачивание автоматически. Для некоторых браузеров сохранение начнётся после стандартного системного диалога.

Ожидание ссылки.
Шаг 1. Вставь ссылку на видео Rutube.
Шаг 2. Нажми кнопку, backend начнёт загрузку на VPS.
Шаг 3. Готовый файл вернётся в браузер и начнёт скачиваться.
